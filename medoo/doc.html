文档

开始之前，我们建议您先阅读指南文档，以便了解如何安装 Medoo 以及开始一个简单的代码。

  • new medoo

    提供开始使用 Medoo 并启动数据库连接的指南

此文章有关于 Medoo 数据过滤，它在大多数 Medoo 函数中都有用到，并以 $where 作为参数，使用它过滤数据以获取我们想要的目标数据，这一点很重要。

  • where

    文档介绍 $where 参数。

以下是 Medoo API 列表。你可找到你想要的任何东西。

  • select

    从数据库中获取数据

  • insert

    插入新记录到表中

  • update

    修改表中的数据

  • delete

    删除表中的数据

  • replace

    替换掉旧数据

  • get

    从表中获取一条记录

  • has

    确定目标数据是否存在

  • count

    计算行数

  • max

    获取列中的最大值

  • min

    获取列中的最小值

  • avg

    获取列平均值

  • sum

    获取列总值

  • query

    插入新记录到表中

  • quote

    转义查询语句中的字符串

  • action

    开始一个事务

  • debug

    输出已生成的 SQL，但不执行。

  • error

    返回最后一次操作中的错误信息

  • log

    返回所有已执行的查询语句。

  • last_query

    返回最后一次查询语句。

  • info

    获取数据库信息。
