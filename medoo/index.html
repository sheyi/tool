喜欢 Medoo?

为什么选择 Medoo

  • 轻量级

    只有 20KB 左右，单文件。

  • 简单

    极易学习和上手，友好的框架结构。

  • 强大

    支持各种常见的或复杂的SQL查询。

  • 兼容

    支持各种 SQL 数据库，包括 MySQL, MSSQL, SQLite, MariaDB, Oracle, Sybase, PostgreSQL 等等。

  • 安全

    防止 SQL 注入。

  • 免费

    基于 MIT 开源协议，如果需要，你可以随时随地使用它。

开始

使用 Composer 安装

$ composer require catfan/Medoo

更新

$ composer update

入门代码

// 如果你是通过 composer 安装，只需在项目的开始部分加上此代码即可自动加载。
require 'vendor/autoload.php';

// 或者你只需下载 medoo.php 并放置到项目目录中，然后 require 即可。
require_once 'medoo.php';

// 初始化
$database = new medoo([
    'database_type' => 'mysql',
    'database_name' => 'name',
    'server' => 'localhost',
    'username' => 'your_username',
    'password' => 'your_password',
    'charset' => 'utf8'
]);

// 开始使用
$database->insert('account', [
    'user_name' => 'foo',
    'email' => 'foo@bar.com',
    'age' => 25,
    'lang' => ['en', 'fr', 'jp', 'cn']
]);
* 在 PHP 5.4+ 中, 你可以使用 [] 作为数组语法。
