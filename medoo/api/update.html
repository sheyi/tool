update

更新表中的数据

update($table, $data, $where)
  • table [string]

    表名称。

  • data [array]

    将要修改的数据。

  • where (optional) [array]

    过滤数据记录的 WHERE 子句。

返回: [number] 受影响的行数。
像 insert() 一样，您可以修改数组数据而不进行序列化，您可以使用 [+], [-], [*] 和 [/] 进行数学运算。
$database->update("account", [
	"type" => "user",

	// 所有 age 加 1
	"age[+]" => 1,

	// 所有 level 减 5
	"level[-]" => 5,

	// 所有 score 乘以 2
	"score[*]" => 2,

	// 像插入一样，你可以进行序列化
	"lang" => ["en", "fr", "jp", "cn", "de"],

	"(JSON) fav_lang" => ["en", "fr", "jp", "cn", "de"],

	// 您还可以分配 # 以使用 SQL 函数
	"#uid" => "UUID()"
], [
	"user_id[<]" => 1000
]);
