last_query

返回执行的最后一个查询。

last_query()
返回: [string]
和 log() 一样，便只返回最后一个查询。
$database->select("account", [
	"user_name",
	"email"
], [
	"user_id[<]" => 20
]);

$database->insert("account", [
	"user_name" => "foo",
	"email" => "foo@bar.com"
]);

echo $database->last_query();
// INSERT INTO "account" ("user_name", "email") VALUES ('foo', 'foo@bar.com')
